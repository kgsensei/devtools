devtools

by kgsensei

All tools are free to use, however, please give credit where it is due.

To view documentation and setup resources just click on the tool you want to use.

Custom CSS

A css library designed to be lightweight and highly customizable. Heavily inspired from Bootstrap, currently running multiple of my websites.

Custom CSS // Extension: Modals

A custom.css extension for modals, designed to work seamlessly with customs.css

Custom CSS // Extension: Inputs

A custom.css extension for inputs, designed to work seamlessly with customs.css

Simple JS

A JS library designed to cut down on the amount of code required to make nice, reactive websites. It's very lightweight and minimal but it gets the job done.

Simple JS // Extension: Animation

A simple.js extension for animation, very similar to jQuery's animation functions.

Theme Builder

A tool dedicated to building nice looking themes quickly, with plenty of suggestions as well.
